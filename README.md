# ToDo App

### ToDo App Features:

  - Add Projects
  - Add Task and owners and mark as done
  - Persistance on local and server

### Structure:
 - TodoApp : Ionic App
 - node-api : API
  
### Run:
 - mongodb
 - node ./node-api/app.js


### Version
0.0.1
